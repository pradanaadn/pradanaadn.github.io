# Portfolio

This portfolio is based on the template from [Oscar Hernandez's Portfolio](https://oscarhernandez.vercel.app/)


## **Project structure**

```
public/
└── svg/
src/
├── Components/
├── layouts/
├── React/
└── pages/
```

## **Local configuration**

1. Clone the repo:

```bash
git clone https://github.com/pradanaadn/pradanaadn.github.io
```

2. Install dependencies:

```bash
npm install
```

3. Start the develpoment server:

```bash
npm run dev
```
